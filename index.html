火车头采集器 高铁采集器 2025年计划任务不能保存 2025年了，火车头采集器也会出现千年虫的bug，这不，计划任务不能保存了。 每日或每周的定时可以正常保存，每间隔多久的就不能保存，提示 End time cannot be before start time。 别慌，后面有个Cron表达式可以暂时解决这个问题， 0 0 0/3 * * ? 表示每间隔3小时执行 0 0 12 * * ?：每天中午 12 点执行。 0 0/5 * * * ?：每 5 分钟执行一次。 0 0 8-18 ? * MON-FRI：周一至周五的 8 点到 18 点之间每小时执行一次。 原文来自 https://www.is96.com/e/287.html 这里有个生成器，以及详细的说明。
